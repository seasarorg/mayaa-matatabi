設定

関連づけ

ファイルの関連づけで拡張子が「mayaa」のファイルにはMayaaエディタを、拡張子が「html」のファイルにはWTPのHTMLエディタが関連づけされている必要があります。

プロジェクトごとの設定

基本・エラーマーカー

プロジェクトの設定画面で「matatabiを使用する」のチェックを入れれば、機能が有効になります。
またMayaaファイルの形式チェックのエラーレベルを変更することが出来ます。

自動生成

自動生成のタブではMayaaファイルの自動生成処理に対する設定が出来ます。 名前空間を追加すれば設定した名前空間がMayaaファイルの生成時に名前空間として設定され、タグ変換ルールではHTMLのタグに対して自動生成時に適用するテンプレートが設定できます。
テンプレートは今のところVelocityを使用しており$をつければ以下のような変数を使うことが来ます。
変数名 型 説明
id String HTMLタグのid属性の値
name String HTMLタグの名前
hasBody boolean HTMLタグが子要素を持っているかどうか
attr Map<String ,org.w3c.dom.Attr> タグについていた属性
